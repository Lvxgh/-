# 个人 AI 记忆系统 · 项目计划摘要

## 项目愿景

做一个本地优先的个人记忆系统：所有数据存在自己电脑上，通过 MCP 协议让任意 AI Agent 共享同一个私人记忆层。口号是"你的 AI 记忆属于你自己"。

## 四个阶段

- 阶段 0：写一个 200 行的本地笔记 RAG 命令行工具，理解 embedding 和向量检索。
- 阶段 1：做成自己每天真实使用的本地 RAG 引擎，支持混合检索和增量索引。
- 阶段 2：记忆系统核心——记忆提取、遗忘策略、冲突解决，并在 LoCoMo 等公开 benchmark 上评测。
- 阶段 3：实现 MCP server，开源发布运营。

## 技术选型

- 语言：Python
- embedding 模型：bge 系列（中英双语，可本地运行）
- 向量存储：先用 numpy 暴力检索，之后再换 sqlite-vec 或 LanceDB
- 本地大模型：Ollama；云端用 Claude API

## 不烂尾原则

每个阶段结束时，产出物必须是自己每天真实在用的东西。
